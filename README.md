# dsa-final-project
Our final project for DSA course at Keiser University. This project simulates the Djikstra and Bellman Ford shortest path algorithms.

For the `venv`, run:
```
python -m venv .venv
```
Activate it with:
**Windows**
```
.\.venv\Scripts\Activate
```
To install the libraries, you can run:
```
pip install -r requirements.txt
```

To exit:
```
pip install -r requirements.txt
```
